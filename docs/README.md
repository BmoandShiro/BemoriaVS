# Documentation

This folder contains all planning documents, system reviews, and implementation guides for the Discord game project.

## Combat System Documentation

- **`COMBAT_MECHANICS_OVERVIEW.md`** - Complete overview of how combat works (damage, accuracy, blocks, dodges, turn order)
- **`COMBAT_SYSTEM_REVIEW.md`** - Review of the combat and enemy handling system with scalability assessment
- **`PARTY_COMBAT_IMPROVEMENTS.md`** - Analysis and recommendations for party combat system improvements

## Damage System Documentation

- **`DAMAGE_VARIABILITY_PLAN.md`** - Plan for adding damage variability to combat (dice rolls, stat multipliers)
- **`HYBRID_COMBAT_SYSTEM_PLAN.md`** - Plan for hybrid combat system combining Elden Ring, OSRS, and D&D mechanics
- **`IMPLEMENT_HYBRID_COMBAT.md`** - Implementation guide for the hybrid combat system

## Database Documentation

- **`DATABASE_REVIEW.md`** - Comprehensive review of SQL structure for quests, NPCs, inventory handling, with ratings and improvements

## Feature Documentation

- **`MINING_LOGIC_DOCUMENTATION.md`** - Documentation of mining logic for the Dank Caverns
- **`WALTS_SHOP_ISSUE.md`** - Documentation of issues and fixes for Walt's Weapons shop

## Notes

- These documents are planning/review documents and may contain historical context
- Some plans may have been implemented, others may be future improvements
- Check individual documents for current status
